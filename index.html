Acerca del proyecto

Plantilla de diseño web adaptable (del ingles responsive web design) para crear aplicaciones de mapas web con Leaflet, código abierto con licencia MIT, disponible en GitHub.

  • TEG - Universidad del Zulia.
  • PHP como lenguaje de programación Backend para servir y gestionar peticiones en el servidor.
  • Iconos Font Awesome
  • Tipografía CSS Roboto Condensed y Nunito de © Google fonts
  • Alertas personalizadas con sweetAlert2
  • Reutilización de código JS Leaflet with a topojson layer
  • Agrupación lógica de marcadores de múltiples capas (marker clustering) a través del leaflet marker cluster plugin.
  • Marcador de punto personalizados con Leaflet.awesome-markers plugin v2.0
  • Botones personalizados con Leaflet.EasyButton
  • Visualización de la capa de © Google Map con el complemento Leaflet GridLayer GoogleMutant
  • Ultima actualización de la capa de establecimientos de salud en Diciembre de 2018
